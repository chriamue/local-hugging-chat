# local-hugging-chat
Docker Compose for local running hugging chat with backend.

## Usage

### 1. Clone this repository

```bash
git clone https://github.com/chriamue/local-hugging-chat
```

### 2. Run docker-compose

```bash
docker-compose up -d
```

### 3. Access to http://localhost:8000
